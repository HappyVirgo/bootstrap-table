## Options

### fixedColumns

- **attribute:** `data-fixed-columns`

- **type:** `Boolean`

- **Detail:**

  set `true` to enable fixed columns.

- **Default:** `false`

### fixedNumber

- **attribute:** `data-fixed-number`

- **type:** Number

- **Detail:**

  the number of fixed columns.

- **Default:** `1`

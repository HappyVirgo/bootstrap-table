Table Export

Use Plugin: tableExport.jquery.plugin

Usage

<script src="extensions/export/bootstrap-table-export.js"></script>

Options

Name Attribute Type Description Default
showExport data-show-export Boolean set true to show export button. true
exportTypes data-export-types Array export types, support types: 'json', 'xml', 'png', 'csv', 'txt', 'sql', 'doc', 'excel', 'powerpoint', 'pdf'. ['json', 'xml', 'csv', 'txt', 'sql', 'excel']
